# Live Stock App

This application tells the user the prices of different stocks, whether there is a rise or fall in prices through numeric and graphical representation.

Steps to run the application:
  - Install node, ruby, compass.
  - Go to the root folder (livestockapp), run 'npm install -g gulp-cli' & 'npm install' in the command prompt.
  - Go to source folder and run 'bower install'
  - That's it. Now you can run the application by going to the root folder (livestockapp) and run command 'gulp'.
